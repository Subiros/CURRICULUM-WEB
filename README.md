# 🌟 Personal CV Website

Welcome to my **personal Curriculum Vitae website**, built with **Astro**.  

This project showcases:  
- **About Me** – a brief introduction  
- **Education** – my academic background  
- **Professional Experience** – my work and projects  
- **CV Download** – download my CV in PDF format  

The site is designed to be **clean, responsive, and easy to navigate**, giving a clear overview of my skills and experience.  

> ⚡ To run locally: just use `npm run dev`.
